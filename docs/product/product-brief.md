# Product Brief

Visual Ideation Studio helps creative users turn a starting image and written prompt into one focused aesthetic reference. The goal is not to flood the user with options, but to make each iteration legible, revisitable, and useful for creative direction.

## Audience

- Designers building campaign, product, or editorial directions
- Portfolio reviewers evaluating AI product thinking
- Creative teams that need fast visual territory exploration

## Core Jobs

- Start from an image and a text direction
- Generate one polished visual reference
- Understand the visual genre and art direction logic
- Compare progress across iterations

## MVP Scope

- Base image upload
- Prompt entry
- One generated outcome
- Tags, palette, references, and rationale
- Local browser timeline

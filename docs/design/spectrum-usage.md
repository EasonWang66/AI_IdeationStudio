# Spectrum Usage

The product should feel like a work-focused creative tool rather than a landing page. React Spectrum components are used for controls, buttons, layout primitives, and accessible interaction states. Custom CSS is limited to product layout and image-heavy surfaces.

## Principles

- Quiet surfaces, dense but scannable panels
- Clear command hierarchy
- Limited radius and restrained color
- Timeline as a production tool, not decorative history
- No Adobe logos or product marks in the UI

## Public Naming

Use **Visual Ideation Studio** publicly. Refer to Adobe only as a design-system inspiration where appropriate, and include a disclaimer in portfolio materials if Adobe trademarks are mentioned.
